【必备条款】当事人名、住所，标的，履行期限、地点、方式，违约责任，争议解法
【标的】合同当事人权利义务指针。合同法律关系客体。须明确、具体、肯定。非非法
【表意行为】意思自治行为。最大量产生民事关系的事实
【补充性规定】民法独有。修补关系残缺
【财产】满足人的需要，能为人控制，稀缺，合法
【财产权】物权、债权、知识产权（兼有人身权）、继承权（兼有人身权）。可转让
【撤回】取消未到达受要约人要约。取消未到达要约人承诺，或撤回与承诺同时到达
【撤销】消灭未承诺已生效要约
【诚实信用】原则。实质为授予法院自由裁量权
【承诺】方式为通知（推荐）、行为
【程序法】保障实体权利实现
【从权】非主权
【缔约责任】违反先合同义务。属过错责任、民事责任
【订立】当事人达成合同主要条款合意
【订约主体】2个或2个以上当事人
【动态财产关系】解决财产流转问题，平等主体由民法调整，不平等主体由经济法、行政法调整
【非表意行为】有事实行为（如埋藏物发现）、非法行为（如侵权）
【格式合同】又标准合同、定式合同、附和合同。格式标准化，式样固定、不可协商，只能接受既定特殊内容。相对人缺乏缔约伙伴完全自由；当事人双方经济地位悬殊，弱者非自愿
【给付】债权客体。债权人行为
【公法】又强行法。规定不可由当事人合意变更。有继承法、民法总则规定、亲属法、物权法
【合同本质】各当事人意思表示一致
【合同成立】民法典条483：承诺生效（通知到达要约人，或作出行为）时
【合同当事人】具有合同订立民事权利、民事行为能力。可委托代理人
【横向财产关系】又民事财产关系。主体地位平等的财产关系
【后合同义务】民法典条558：合同终止后，当事人所承担法定义务
【加工承揽】A义务：加工、交付标的，A权利：收取加工费，B义务：支付加工费，B权利：接受标的
【价金】又报酬（标的为行为）、价款（标的为物）。当事人A履行义务时，B以货币形式支付的代价。有政府定价、政府指导价
【禁止流通物】如矿藏、水流
【静态财产关系】民法调整，解决财产归属问题，财产支配者地位平等，依法行使财产权利
【绝对权】特定人权利，一切人义务。如人格权、物权、知识产权
【抗辩权】对抗请求权的权。有一时抗辩权（如不安抗辩权、同时履行抗辩权）、永久抗辩权（如时效届满拒履义务抗辩权）
【可归责性】一方存在过错。如：民法典条500
【立法规制】立法手段规制格式合同不公平条款。有特别规制、一般规制
【买卖】A义务：转移标的所有权，A权利：收取货款，B义务：支付货款，B权利：取得标的所有权
【民法】调整平等民事主体间财产、人身关系的法律规范。实体法。商品经济基本法。规定民法权利
【民法典】成文民法最高形式
【民法三要素】客体、内容、主体
【民事关系】主体地位平等，大多依民事主体意志，权利义务对等相互，补偿性保障
【民事救济】事后调整手段。恢复破坏权利
【民事客体】物、行为（一些）、智力成果、人身利益（有争议）
【民事权利】人格权、身份权、物权、债权、知识产权、继承权。一权一关系
【名称】法人、组织在登记机关登记的正式称谓
【默示形式】无语言，无行为。不构成意思表示
【请求权】请求他人为或不为一定行为。如物请求、债请求
【人】自然人、法人、合伙
【人身权】人格权、身份权、知识产权（兼有财产权）、继承权（兼有财产权）。不可转让
【生效地】有作出对话地，通知到达地，行为完成地
【实体法】规定主体权利义务
【书面形式】民法典条469：可有形表现内容的形式，如合同书、信件、数据电文（电报、电传、传真、电子数据交换、电子邮件）。采用最为普遍：法定采用、约定采用
【司法规制】司法机关否定、限制格式合同不公平条款
【私法】规定可由当事人合意变更
【特定物】具有独立特征、不可代替性的物
【同命运】主权被设定负担、消灭、转移，从权随之设定负担、消灭、转移
【推定形式】无语言，有行为
【先合同义务】要约生效后、合同生效前，当事人所承担法定义务。属注意义务。非给付义务
【限制流通物】如白银、黄金、外汇
【宪法】规定政治权利
【相对权】特定人权利义务。如身份权、债权、继承权
【协商一致】协议达成须合同主要条款协商一致
【信赖利益】合同有效存在，利益数额不大于履行利益
【行为】权利人行使权利，义务人履行义务
【行为模式】事前调整手段。假定+处理。假定=民法适用条情。处理=可做或禁做或应做。有任意（自选）、强制
【行政规制】政府权力认可、许可、核准、监督格式合同
【形成权】单方意思表示。如撤销权、抵销权、解除权、终止权、追认权
【要约】民法典条472：要约人希望和受要约人订立合同的意思表示，内容明确具体
【要约失效】拒绝通知到达要约人，要约人依法撤销，受要约人未承诺，受要约人变更内容，当事人丧失民事行为能力
【要约邀请】合同订立预备（事实）行为，无法律意义
【遗嘱继承】事实构成：被继承人死亡、被继承人遗嘱、继承人接受继承
【意思自治】原则。民事主体依意愿协商设立、变更、终止民事关系，意思表示先于推定条款
【争议解法】有当事人协商和解、第三方调解、诉讼、仲裁
【支配权】人支配客体。排他性。如人身权、物权、知识产权
【种类物】具有共同物理属性、可代替性的物
【主权】居主导地位、可独立存在的权
【住所】公民长久居住场所，法人、组织主要办事机构所在地
【纵向财产关系】主体地位不平等的财产关系
